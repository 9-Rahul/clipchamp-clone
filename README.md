# Clipchamp Clone

A modern, responsive clone of the Clipchamp landing page built with *Next.js* and *React*.

# Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://reactjs.org/)
- **Styling**: Vanilla CSS (CSS Modules)
- **Typography**: Inter (via `next/font/google`)

# Key Features
- **Responsive Header**: Dynamic navigation with custom hover effects and dropdowns.
- **Mobile Sidebar**: A smooth, slide-out navigation drawer for mobile users.
- **Modular Components**: Reusable UI components like Buttons, Header, and Navbar.
- **Centralized Design**: Global CSS variables for consistent brand colors and typography.
